# Dice Overview
The dice can be enhanced in various ways. Here is the list of everything you can do to a dice.

## Dice enhancements 
Dice enhancements are basically card enhancements in Balatro. The only main difference is that blurry dice take the place of wild cards.

- Bone Dice + Add 4 mult
- Lucky Dice + chance at money and mult
- Wooden Dice + Add 10 base miles
- Iron Dice + x1.5 mult when held in hand
- Gold Dice + Add money when kept
- Loaded Dice + double the odds of one side being rolled (You pick the side) - before roll will always show loaded side up with a circle around the pips
- Diamond Dice + x2mult and 25% chance of cracking
- Stone Dice + no pips, counts as 50 base miles
- blurry - counts as any number in a straight

## Pip enhancements
Pips are like seals in Balatro. I've even retained the same seal colors to make it easy.

- purple flower pips - if a purple pips land face up get a supply card (must have room)
- red bullet pips - if a red pip lands face up trigger dice twice
- golden dollar sign pips - if a gold pips land face up get $3 when scored
- blue moon pips - if a blue pips land face up hold on to for trail guide at end of game

## Dice Aura
Instead of card editions like Balatro, we have dice Aura, which behaves mostly the same.

- holy aura - x1.5 mult
- fire -  +10 mult
- icy - + 50 pips
